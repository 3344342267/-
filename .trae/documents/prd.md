# AI写作助手 - 产品需求文档

## 1. 产品概述

### 产品名称
**墨韵 - AI智能写作平台**

### 产品定位
一款面向小说创作者的AI辅助写作工具，支持连接用户自选API，支持多模型提供商，提供从大纲到章节的完整写作流程。

### 核心价值
- 简化小说创作流程，从大纲到正文一站式完成
- 支持多种主流AI模型，灵活适配用户需求
- 智能学习参考作品风格，不复制剧情

## 3. 布局结构

### 整体布局
```
┌─────────────────────────────────────────┐
│           顶部状态栏 (可折叠)             │
├──────────┬─────────────────────┬──────────┤
│          │                     │          │
│   左侧   │                     │   右侧   │
│  侧边栏   │      主内容区域       │  侧边栏  │
│  作品列表  │                     │   设置   │
│          │                     │          │
├──────────┴─────────────────────┴──────────┤
│           底部工具栏 (可折叠)             │
└─────────────────────────────────────────┘
```

### 响应式策略
- **移动优先**: 所有布局基于移动端设计
- **断点**: 
  - < 768px: 手机端，单列布局
  - 768px - 1024px: 平板端，双列可选
  - > 1024px: 桌面端，三列布局

### 侧边栏设计
- **左侧作品栏**: 宽度280px，点击切换显示/隐藏
- **右侧设置栏**: 宽度320px，点击切换显示/隐藏
- **移动端**: 全屏覆盖，点击外部或X按钮关闭

## 4. 功能模块

### 4.1 API设置模块

#### 模型供应商选项
- **Google Gemini API 兼容**
- **Claude API 兼容**
- **OpenAI Responses API 兼容**
- **OpenAI API 兼容**
- **自定义 API** (用户输入地址)

#### 配置项
1. **模型供应商选择**: 下拉菜单
2. **API 地址**: 输入框 (可选，自动填充默认值)
3. **API 密钥**: 密码输入框
4. **模型名称**: 输入框 + 快捷获取按钮
5. **获取模型列表按钮**: 全屏显示加载状态
6. **测试连接按钮**: 验证配置是否正确

#### 模型获取流程
1. 用户填写API配置
2. 点击"获取模型"按钮
3. 显示加载动画
4. 返回可用模型列表
5. 点击模型名称 → 自动填充到模型输入框
6. 点击"测试连接" → 验证并弹出结果

#### UI设计
- 圆角卡片设计
- 下拉箭头点击弹出模型列表
- 成功/失败弹窗提示

### 4.2 写作工作流

#### 第一步：创建大纲
- **输入项**:
  - 书名 (可选)
  - 主角/配角设定
  - 章节数
  - 每章剧情走向 (可多行输入)
  - 写作风格参考 (可选)
- **功能**:
  - 根据剧情自动生成大纲
  - 手动输入/编辑大纲
  - 大纲复制/重新生成
  - 市面热门剧情推荐

#### 第二步：创建书籍
- 创建新书 → 输入书名
- 自动汇总大纲到书籍信息
- 支持从书院搜索参考书

#### 第三步：章节生成
- 当前章节内容输入
- 字数约束设置
- 生成按钮 → AI生成内容
- 停止按钮 → 停止生成
- 继续生成 → 追加内容
- 重写/编辑 → 修改AI输出
- 撤回 → 回退AI消息

#### 第四步：作品集
- 采纳生成内容 → 存入作品集
- 每章自动总结到大纲
- TXT格式导出
- 下载/复制功能

### 4.3 参考资料模块

#### 书源功能
- 内置书源: `https://shuyuan.nyasama.cc/shuyuan/f5b15e9641d164937061974cfefb675c.json`
- 搜索书籍 → 查看详情
- 学习风格 (除剧情外全学)

#### 上传参考
- 上传文件 (txt, md, pdf)
- 上传图片
- 输入URL爬取内容

#### 学习方式
- 提取文风/手法
- 不学习剧情走向
- 可混合使用多个参考

### 4.4 预设管理

#### 预设格式
支持酒馆/NovelAI格式
```json
{
  "title": "预设名称",
  "description": "描述",
  "content": "预设内容"
}
```

#### 功能
- 上传预设文件
- 保存当前设置到预设
- 加载/应用预设

### 4.5 历史记录

#### 功能
- 保存每次对话
- 点击切换历史
- 删除历史
- 重命名历史

### 4.6 对话气泡

#### 用户消息
- 右侧气泡，背景色 #E8F5E9
- 支持Markdown渲染

#### AI消息
- 左侧气泡，背景色 #F5F5F5
- 支持Markdown渲染
- 显示思维链 (可折叠)
- 可编辑/撤回

## 5. 组件清单

### 5.1 侧边栏组件
- **侧边栏容器**: 背景、圆角、阴影
- **折叠按钮**: 圆形、图标
- **内容区**: 滚动、自适应高度

### 5.2 按钮组件
- **主按钮**: 墨绿色、圆角
- **次按钮**: 描边、无背景
- **图标按钮**: 圆形、透明背景
- **危险按钮**: 红色

### 5.3 输入组件
- **文本输入框**: 圆角、聚焦高亮
- **文本域**: 自动高度、滚动
- **下拉选择器**: 自定义样式
- **密码输入**: 显示/隐藏切换

### 5.4 卡片组件
- **设置卡片**: 圆角、阴影、分组
- **作品卡片**: 封面、标题、章节数
- **章节卡片**: 标题、字数、状态

### 5.5 弹窗组件
- **模态弹窗**: 居中、遮罩
- **侧滑弹窗**: 从底部滑入
- **Toast提示**: 顶部居中、自动消失

### 5.6 折叠组件
- **可折叠标题**: 箭头图标、点击展开
- **可折叠内容**: 高度动画
